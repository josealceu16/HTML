											<!--Imagens-->
											
											
										
Vamos usar tags que são feitas para trabalhar com jpg, jpeg, png, gif's... entre outros tipos de imagens.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------

<p>
    <img src="">
</p>

--Não precisamos colocar a nossa tag img dentro de uma tag p.. aí eu coloquei para que ficasse melhor de fazer um modo slide;
--Para pegarmos uma imagem utilizamos dessa maneira...
--O src é o local onde vamos colocar um link que nos levará onde está a imagem que nós queremos usar... se formos puxar uma imagem do browser, por exemplo, usaremos um link ansoluto, 
se formos buscar em uma pasta usaremos links relativos.
