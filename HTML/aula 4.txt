E dentro das aspas colocamos a tag que vai ser utilizada para selecionar os link, no caso seria a tag 'a'

					// mas o nosso objetivo é pegar o link fazer uma requisição em ajax e juntar tudo, só vai acontecer dessa maneira//
								
								
								
-- para realizarmos essa requizição vamos usar o nosso comando 'feth(link.href) para fazer a nossa requisição ajax;
-- e precisamos tbm pegar o nosso conteudo, a partir do comando => ------ const conteudo = document.getElementById('conteudo') ----------- dentro das aspas colocamos os nome que demos para a nossa section que nele esta contido uma 'id';

-- criaremos um atributo chamado then e nele colocaremos uma função chamada resp.text para pegar o meu conteudo html;
-- e mais uma vez criaremos o then que dessa vez vai usar a função html para que pegar todo o conteudo html que esta em html;


O meu código vai ficar dessa maneira: 



  <script>
    document.querySelectorAll('a').forEach(link => {
        const conteudo = document.getElementById('conteudo')
        link.onclick = function (e) {
            e.preventDefault()
            fetch(link.href)
                .then(resp => resp.text())
                .then(html => conteudo.innerHTML = html)
        }
    })
</script>
